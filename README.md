# Analyse-der-Videospielverk-ufe
Dieses Projekt untersucht die Verkaufszahlen von Videospielen im Zeitraum 2014–2016. Ziel ist es, Markttrends zu erkennen, stabile Segmente zu identifizieren und konkrete Handlungsempfehlungen für das Management von GameCo abzuleiten. Die Analyse basiert auf regionalen, genrebezogenen und plattformbezogenen Verkaufsdaten.
